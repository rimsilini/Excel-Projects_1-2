## Project -1 - Title: Global Defense Systems Investment Analysis (Excel Project)

---

## Introduction  
📊 This project analyzes global defense technology investments using Excel, focusing on funding distribution, dominant technology domains, and country-level investment patterns.

---

## Background  
This project was created to explore how defense funding is allocated across different technology domains and countries, using structured Excel analysis to extract clear, data-driven insights.

---

## The questions I wanted to answer through my Excel analysis were:

1. How is funding distributed across defense projects?  
2. Which technology domains are most common?  
3. Which domains receive the highest average funding?  
4. Which countries invest the most overall?  
5. How do volume and funding compare across domains?  

---

## Tools I Used  

- **Microsoft Excel**: Data analysis and structuring  
- **Pivot Tables**: Aggregation by country and domain  
- **Basic Statistics**: Averages, totals, distributions  
- **Git & GitHub**: Project documentation and version control  

---

## The Analysis  

---

### 1. Funding Distribution Across Defense Projects  

- **Total Projects:** 200  
- **Average Funding:** $62,477,500  
- **Minimum Funding:** $26,000,000  
- **Maximum Funding:** $120,000,000  
- **Median Funding:** $60,250,000  

#### Key Insights:
- 50% of projects fall between **$41,750,000 and $84,000,000**  
- Funding distribution is relatively concentrated with no extreme outliers  

---

### 2. Most Dominant Defense Technology Domains  

| Tech Domain          | Project Count |
|---------------------|--------------|
| AI & Autonomy       | 53           |
| Missile Defense     | 49           |
| Cyber Defense       | 36           |
| Space Systems       | 34           |
| Surveillance & ISR  | 28           |

#### Key Insights:
- AI & Autonomy has the highest number of projects  
- Missile Defense closely follows in volume  

---

### 3. Funding by Technology Domain  

| Tech Domain          | Avg Funding ($) |
|---------------------|----------------|
| Missile Defense     | 75,734,693     |
| Space Systems       | 61,985,294     |
| Surveillance & ISR  | 61,642,857     |
| AI & Autonomy       | 58,518,868     |
| Cyber Defense       | 51,375,000     |

#### Key Insights:
- Missile Defense has the highest average funding per project  
- Cyber Defense has the lowest average funding among the listed domains  

---

### 4. Country-Level Investment Analysis  

| Country | Total Funding ($) |
|--------|------------------|
| France | 3,562,000,000    |
| USA    | 2,529,500,000    |
| Canada | 2,295,000,000    |
| UK     | 1,200,000,000    |
| Poland | 607,500,000      |

#### Key Insights:
- France has the highest total funding in the dataset  
- The USA and Canada follow with substantial investment levels  

---

### 5. Strategic Positioning: Volume vs Funding  

- AI & Autonomy → Highest project volume  
- Missile Defense → Highest average funding  
- Cyber Defense → Lower average funding compared to other domains  
- Space Systems → Mid-range in both volume and funding  

---

## What I Learned  

- How to structure and analyze datasets using Excel  
- How to use pivot tables for aggregation and comparison  
- How to interpret funding distributions and domain-level trends  
- How to translate raw data into structured insights  

---

## Conclusions  

### Insights  

- Missile Defense leads in average funding per project  
- AI & Autonomy dominates in project count  
- France is the top investor in total funding within this dataset  
- Funding distribution is relatively consistent across projects  

---

## Closing Thoughts  

This project demonstrates how Excel can be used to analyze structured datasets and extract meaningful insights from funding and technology trends within the defense sector.

## Project -2- Title: Defense Systems Scenario & Performance Analysis (Excel Project)

---

## Introduction  
📊 This project analyzes defense system performance and cost scenarios using Excel, focusing on regional performance metrics, scenario-based cost adjustments, and pivot-driven insights.

---

## Background  
This project was developed to evaluate how defense system performance and costs vary across regions and under different simulated scenarios.

The analysis relies on structured Excel modeling, including pivot tables, validation layers, and scenario multipliers to explore how key variables impact outcomes.

---

## The questions I wanted to answer through my Excel analysis were:

1. How do performance metrics vary across regions?  
2. What differences exist between regions in terms of response time and success rate?  
3. How are costs impacted under different scenario conditions?  
4. What role do scenario multipliers play in adjusting total cost?  
5. How can pivot tables be used to summarize regional performance?  

---

## Tools I Used  

- **Microsoft Excel**: Core tool for modeling and analysis  
- **Pivot Tables**: Aggregated regional performance metrics  
- **Scenario Analysis**: Evaluated cost variations under different assumptions  
- **Data Validation**: Ensured structured and controlled inputs  
- **Dashboarding**: Visual representation of key metrics  
- **Git & GitHub**: Documentation and version control  

---

## The Analysis  

Each part of the analysis focuses on structured outputs directly derived from Excel sheets, including pivots and scenario models.

---

### 1. Regional Performance Analysis (Pivot Tables)  

Using pivot tables, performance metrics were grouped by region.

#### Observations:
- Regions included in the analysis: **APAC, EU, ME**  
- Metrics analyzed include:
  - Success Rate  
  - Response Time (ms)  
  - Additional operational indicators from the dataset  

#### Key Insights:
- Performance varies by region based on response time and success rate  
- Each region shows distinct operational characteristics in the pivot output  

---

### 2. Scenario-Based Cost Analysis  

Scenario modeling was used to simulate how total costs change under different conditions.

#### Observations:
- A **baseline scenario** is defined in the model  
- Additional scenarios apply **multipliers** to adjust total cost  
- Scenario outputs include adjusted total cost values  

#### Key Insights:
- Scenario multipliers directly impact total cost outputs  
- The model allows comparison between baseline and adjusted scenarios  

---

### 3. Scenario Multipliers  

A dedicated sheet defines multipliers used in scenario simulations.

#### Observations:
- Multipliers are applied to cost components  
- Different scenarios apply different scaling factors  

#### Key Insights:
- Cost sensitivity is controlled through multiplier adjustments  
- This enables structured simulation rather than arbitrary changes  

---

### 4. Program Cost Modeling  

The model includes a structured cost calculation layer.

#### Observations:
- Costs are calculated using predefined inputs and formulas  
- Outputs are reused in scenario analysis  

#### Key Insights:
- The model separates raw data, calculations, and scenario outputs  
- This structure improves clarity and traceability  

---

### 5. Dashboard & Visualization  

A dashboard sheet is used to present key outputs.

#### Observations:
- Visual elements summarize performance and cost data  
- Metrics from pivots and scenarios are displayed for interpretation  

#### Key Insights:
- Dashboard consolidates multiple analysis layers into a single view  
- Enables quick comparison across regions and scenarios  

---

## What I Learned  

- How to structure multi-layer Excel models (raw data → calculations → scenarios → dashboard)  
- How to use pivot tables to summarize regional performance  
- How to build scenario analysis using multipliers  
- How to separate inputs, logic, and outputs for clarity  

---

## Conclusions  

### Insights  

- Regional performance varies across key operational metrics  
- Scenario analysis provides a structured way to simulate cost changes  
- Multipliers are an effective method for modeling uncertainty  
- Excel can support multi-layer analytical models combining performance and financial analysis  

---

## Closing Thoughts  

This project demonstrates how Excel can be used not just for analysis, but for building structured analytical models that combine performance metrics and scenario-based forecasting.

The approach reflects a more advanced use of Excel, integrating data, logic, and simulation into a single workflow.
